# 线程

首先你应该有平行宇宙的概念. 可能有些中二病少年会想到平行宇宙, 我们身处这个宇宙, 还有一个与我们所在宇宙并存的一个宇宙, 两个宇宙各自在上演着各自的故事......

Java中也有这样的概念, 叫做“线程”.

while(true){
    System.out.println("hello");
}

while(true){
    System.out.println("hi");
}

现在存在一个问题, 我想让hello和hi都同时运行, 可是第一个死循环把代码卡住了, 导致下面的死循环执行都没被执行. 这该怎么办?

方法很简单, 我们创建两个线程分别执行这两个方法:

new Thread() {
	@Override
	public void run() {
		while(true)
			System.out.println("hello");
	}
}.start();

new Thread() {
	@Override
	public void run() {
		while(true)
			System.out.println("hi");
	}
}.start();

我们创建了两个Thread对象, 这是简写写法, 你也可以写一个类, 继承Thread, 覆写run方法, 再创建那个类的对象并调用该对象的start方法, 只不过有点麻烦.

运行你会发现, hello和hi都会被输出.
